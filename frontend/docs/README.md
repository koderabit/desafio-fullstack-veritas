# Documentação de Fluxos - Mini Kanban

Esta pasta contém os fluxos de usuário e de dados do projeto "Mini Kanban - Veritas". 
Os diagramas ajudam a compreender como o usuário interage com o sistema e como os dados são processados pelo backend.

## Arquivos

### user-flow.png
Diagrama que mostra o fluxo de interação do usuário com a aplicação:
- Adiciona uma tarefa
- Move tarefas entre colunas
- Exclui tarefas

### data-flow.png
Diagrama opcional que representa o fluxo de dados entre frontend e backend:
- Frontend envia requisições ao backend
- Backend processa dados e retorna respostas

## Como visualizar
Basta abrir os arquivos PNG em qualquer visualizador de imagens ou editor que suporte o formato.
